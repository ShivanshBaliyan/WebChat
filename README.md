# WebChat
# FastAPI WebSocket Chat Application

A real-time chat application built using FastAPI, WebSockets, HTML, CSS and JavaScript.

## Features

- Real-time messaging
- Multiple users
- Responsive UI
- Username support
- WebSocket communication

## Tech Stack

- FastAPI
- WebSockets
- HTML
- CSS
- JavaScript
